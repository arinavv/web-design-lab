1 Структура базового HTML-документа и какие теги обязательно должны быть включены в него?

Обязательные теги:

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Название страницы</title>
</head>
<body>
Содержимое страницы
</body>
</html>

Минимальная структура:

  • <!DOCTYPE html> - объявление типа документа
  • <html> - корневой элемент
  • <head> - служебная информация
  • <title> - заголовок страницы
  • <body> - видимое содержимое
Примечание: Тег <meta charset="UTF-8"> также рекомендуется включать для корректного отображения символов.
2 Какой формат изображений нельзя использовать для размещения на HTML странице: JPEG, PNG, GIF, RAW, SVG, WEBP?

RAW - этот формат нельзя использовать для размещения на HTML-странице.

Причина: Браузеры не поддерживают отображение RAW-файлов (сырые данные с матрицы фотоаппарата).

Поддерживаемые форматы:

JPEG
PNG
GIF
RAW
SVG
WEBP
Информация: SVG (Scalable Vector Graphics) — векторный формат, остальные — растровые форматы.
3 На какой версии языка программирования HTML Вы писали лабораторную работу №1?

HTML5

HTML5 — это пятая и текущая основная версия языка разметки HTML. Она была выпущена в 2014 году и включает множество новых семантических элементов, API для сложных веб-приложений и улучшенную поддержку мультимедиа.

<!DOCTYPE html> <!-- Объявление HTML5 -->
<html lang="ru"> <!-- Атрибут lang для указания языка -->
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Мой сайт</title>
</head>
4 Какой тег в HTML используется для вставки разрыва строки?

<br> - тег для переноса строки.

Пример использования:

<p>Первая строка<br>
Вторая строка<br>
Третья строка</p>

Результат:

Первая строка
Вторая строка
Третья строка
Примечание: Тег <br> является одиночным (самозакрывающимся) тегом и не имеет закрывающего тега. В XHTML пишется как <br />.
5 Если Вы написали код страницы в текстовом редакторе или редакторе кода и запустили полученный файл, смогут ли другие пользователи сети Интернет найти её, вставив в поисковую строку её адрес?

Нет, другие пользователи интернета не смогут найти эту страницу.

Причины:

  • Страница находится только на вашем локальном компьютере
  • У нее нет публичного URL-адреса в интернете
  • Она не размещена на веб-сервере
  • Поисковые системы не индексируют локальные файлы

Чтобы сделать страницу доступной, нужно:

  1. Приобрести доменное имя (например: mysite.com)
  2. Арендовать хостинг (веб-сервер)
  3. Загрузить файлы на сервер через FTP или файловый менеджер
  4. Настроить DNS-записи для связи домена с хостингом
Альтернатива: Можно использовать бесплатные хостинги (GitHub Pages, Netlify, Vercel) для размещения статических сайтов без покупки домена и хостинга.

Локальный путь vs Веб-адрес:

Локальный путь: file:///C:/Users/Имя/Документы/моя_страница.html
Веб-адрес: https://www.mysite.com/моя_страница.html
